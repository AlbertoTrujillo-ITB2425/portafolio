Monitor de Recursos Shell

Bash

Script para monitorizar recursos del sistema en tiempo real con alertas por email cuando se alcanzan umbrales críticos.

#!/bin/bash
# monitor_recursos.sh
# Monitoriza CPU, Memoria y Disco
THRESHOLD_CPU=90
THRESHOLD_MEM=85
THRESHOLD_DISK=90

# Comprobar uso de CPU
CPU_USAGE=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}' | cut -d. -f1)
if [ "$CPU_USAGE" -gt "$THRESHOLD_CPU" ]; then
    echo "ALERTA: Uso de CPU al $CPU_USAGE%" | mail -s "Alerta de Servidor" admin@example.com
fi

Backup Automatizado

Python

Script de Python para realizar backups automáticos de bases de datos MySQL y archivos críticos, con rotación de copias y compresión.

#!/usr/bin/env python3
# backup_script.py
import os
import datetime
import subprocess
import shutil

# Configuración
BACKUP_DIR = "/backups"
DB_NAME = "mi_base_datos"
KEEP_DAYS = 7

# Generar nombre de archivo con fecha
date_str = datetime.datetime.now().strftime("%Y%m%d_%H%M%S")
backup_file = f"{BACKUP_DIR}/db_{DB_NAME}_{date_str}.sql.gz"

# Ejecutar backup de MySQL
subprocess.run(f"mysqldump {DB_NAME} | gzip > {backup_file}", shell=True)
